PriceTrail

In Progress

Web scraper that collects product prices from Kabum's API, stores the historical data in PostgreSQL, and exposes it via a REST API for external consumption. The project is containerized with Docker and includes a scheduler for automated runs.

Key decisions: FastAPI was chosen to expose the data as a structured, queryable endpoint rather than leaving it locked in a database. Docker ensures the environment is reproducible across machines.

Olist dbt Pipeline

Complete

End-to-end data pipeline for the Brazilian e-commerce dataset (Olist / Kaggle). Raw CSV files are ingested into BigQuery via Python, then transformed through a full dbt layer organized as staging → intermediate → marts, grouped by business domain (commercial, logistics, customer experience).

Key decisions: Mart organization by domain reflects how data teams structure models for different stakeholders. CI/CD via GitHub Actions with Workload Identity Federation means the pipeline validates and deploys automatically on every push.

Finance Data Warehouse

Complete

Dimensional model for a fintech transaction dataset focused on fraud detection analysis. The project defines grain, surrogate keys, conformed dimensions, and implements a star schema (fact_transactions, dim_customer, dim_merchant, dim_date) in PostgreSQL.

Key decisions: Scope was deliberately limited to modeling and schema design — the analytics layer sits on top and is out of scope. Business questions, assumptions, and out-of-scope queries are fully documented in the README.
